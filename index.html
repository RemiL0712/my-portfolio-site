Проєкти

Flask Task Manager

Повноцінний менеджер завдань на Flask з використанням SQLite та SQLAlchemy. Додаток дозволяє користувачам додавати, редагувати, видаляти та позначати завдання як виконані. Інтерактивність інтерфейсу забезпечена за допомогою JavaScript.

Python File Sorter

Автоматизований інструмент, написаний на Python, що розпізнає та сортує файли різних типів (зображення, документи, архіви) по відповідних папках. Це рішення для ефективної організації файлового простору та підвищення продуктивності.

Переглянути на GitHub

Веб-скрейпер на Flask

Веб-додаток на Python та Flask для динамічного збору даних з вебсайтів. Застосунок дозволяє користувачам вводити URL, після чого збирає та відображає структуровану інформацію в зручному табличному форматі. Включає надійну обробку помилок.
